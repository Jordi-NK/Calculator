# Calculator

A modern scientific calculator built using HTML, CSS, and JavaScript.
It supports basic arithmetic, trigonometric functions, square and cube roots,
and advanced mathematical operations.

## ✨ Features

- Basic operations (+, -, ×, ÷)
- Responsive design
- Error handling (e.g. division by zero)
- Functions like (sin, cos, tan)
- Added the DEG and RAG toggle button
- Added square roots
- Added exponents

-Etc.....

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
